# vihanti.com

This is the source for [vihanti.com](https://www.vihanti.com).

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

To get a code size report, use:

```
yarn build --report
```

### Lints and fixes files
```
yarn lint
```
